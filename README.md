# ERP Sistema de Gestão

Sistema ERP completo desenvolvido com **React + Vite + TailwindCSS**.

## Módulos

- **Dashboard** — KPIs, gráficos de receita/despesa, produtos mais vendidos, alertas
- **Vendas** — Gestão de pedidos com itens, status, pagamento e histórico
- **Financeiro** — Contas a pagar e a receber, controle de vencimentos, saldo previsto
- **Estoque** — Cadastro de produtos, margem de lucro, alertas de estoque mínimo
- **Clientes** — Cadastro de clientes PF/PJ, histórico de compras
- **Fornecedores** — Cadastro e gestão de fornecedores
- **Relatórios** — Gráficos de desempenho, análise financeira, alertas de estoque

## Tecnologias

- React 19 + React Router DOM
- Vite 8
- TailwindCSS 3
- Recharts (gráficos)
- Lucide React (ícones)

## Como executar

```bash
cd erp-app
npm install
npm run dev
```

Acesse em: http://localhost:3000
